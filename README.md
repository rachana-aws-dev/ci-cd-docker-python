# CI/CD Pipeline for Dockerized Python App

##  Overview
This project demonstrates a CI/CD pipeline using GitHub Actions.

## Tech Stack
- Python (Flask)
- Docker
- GitHub Actions

## CI/CD Workflow
- Code push triggers pipeline
- Installs dependencies
- Runs tests
- Builds Docker image

## Project Structure
app.py  
requirements.txt  
Dockerfile  
.github/workflows/ci-cd.yml
